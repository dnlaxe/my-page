Part 1

The Gist

Given a list of instructions for turning a 100 position circular safe dial (numbered 0 to 99), track the position of the dial as we spin it left and right, and count how many times it lands on exactly 0.

Notes on Modulus on Negative Numbers

In Go, % doesn’t mean modulo, it means remainder. Because of this, it cannot be used directly to find the modulus of negative numbers.

For example, -1 % 5 mathematically should be 4, but Go will evaluate it to be -1 because it truncates towards 0.

To solve this, first evaluate the expression and see whether the result is negative or positive (including the 0 case).

If it is positive, the answer is correct. If it is negative, add the divisor, which wraps the answer back into a positive value.

For example, -1 % 5 evaluates to -1, which is negative, so we add the divisor (-1 + 5) to get the correct answer of 4.

func modulo(a, b int) int {
    r := a % b
    if r < 0 {
        return r + b
    }
    return r
}

Simplifying further, we can also just unconditionally add b to the initial remainder, as we will be easily able to wrap it back into bounds when we apply the modulus again.

This means we no longer need to use branching.

In other words,

  • a % b: The initial remainder.

  • (a % b) + b: Adding b ensures a negative result becomes positive, and a positive result gets pushed forward beyond the next boundary.

  • ((a % b) + b) % b: The final modulo has no effect on the originally negative result, but safely wraps the positive result back into the correct boundary.

func wrap(a, b int) int {
	return ((a % b) + b) % b
}

My final code

package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
)

type Simulation struct {
	Moves           []int
	InitialPosition int
	TotalPositions  int
	Target          int
}

func main() {
	moves, err := ParseInstructionsFromFile("input.txt")
	if err != nil {
		fmt.Println("Failed to parse instructions: ", err)
		return
	}

	sim, err := NewSimulation(moves, 50, 100, 0)
	if err != nil {
		fmt.Println("Simulation could not be created:", err)
		return
	}

	result, err := sim.Run()

	if err != nil {
		fmt.Println("Simulation failed: ", err)
		return
	}

	fmt.Println(sim.Target, "was hit", result, "times.")
}

func NewSimulation(moves []int, start, total, target int) (*Simulation, error) {
	if total <= 0 {
		return nil, fmt.Errorf("Total positions must be greater than 0")
	}

	return &Simulation{
		Moves:           moves,
		InitialPosition: start,
		TotalPositions:  total,
		Target:          target,
	}, nil
}

func ParseInstructionsFromFile(filename string) (movesList []int, err error) {
	file, err := os.Open(filename)
	if err != nil {
		return nil, fmt.Errorf("Failed to open file: %v", err)
	}
	defer file.Close()

	movesList = []int{}
	scanner := bufio.NewScanner(file)

	for scanner.Scan() {
		line := scanner.Text()

		if len(line) == 0 {
			continue
		}

		distance, err := strconv.Atoi(line[1:])
		if err != nil {
			return nil, fmt.Errorf("line %s: %w", line, err)
		}

		switch line[0] {
		case 'R':
			movesList = append(movesList, distance)
		case 'L':
			movesList = append(movesList, -distance)
		default:
			return nil, fmt.Errorf("invalid direction '%c' on line: %s", line[0], line)
		}
	}

	if err := scanner.Err(); err != nil {
		return nil, fmt.Errorf("scan: %w", err)
	}

	return movesList, nil

}

func (s *Simulation) Run() (int, error) {

	count := 0
	currentPosition := s.InitialPosition

	for _, move := range s.Moves {
		position := currentPosition + move
		currentPosition = s.Wrap(position)

		if currentPosition == s.Target {
			count++
		}
	}
	return count, nil
}

func (s *Simulation) Wrap(position int) int {
	return ((position % s.TotalPositions) + s.TotalPositions) % s.TotalPositions
}

Part 2

The Gist

Given a sequence of instructions for turning a 100 position circular dial, calculate the password by totaling every time the dial passes through 0, including full rotations and partial movements that cross the zero mark during the transition, rather than just checking its final landing spot.

Updated Run function

func (s *Simulation) Run() (int, error) {
	count := 0
	currentPosition := s.InitialPosition

	for _, move := range s.Moves {
		distance := move
		if distance < 0 {
			distance = -distance
		}

		fullTurns := distance / s.TotalPositions
		remainder := distance % s.TotalPositions

		count += fullTurns

		if move > 0 {
			if currentPosition+remainder >= s.TotalPositions {
				count++
			}
		} else {
			if remainder != 0 && currentPosition != 0 && currentPosition-remainder <= 0 {
				count++
			}
		}

		currentPosition = s.Wrap(currentPosition + move)
	}

	return count, nil
}

Full turns shows how many times it crosses zero. But it is also important to consider the remainder as that may too be enough to push the dial past zero.

If the move was positive and it’s remainder pushes the position past 99, one more is added.

If the move is negative, leaves a remainder, is not currently on zero and it’s current position with remainder subtracted is less than zero, the dial has crossed zero so one more should be added.
